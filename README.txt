rechtstexte24, aktuelle Rechtstexte von spezialisierten Anwälten!

== Description ==

rechtstexte24 ist der Dienst, welcher Ihre Seite immer mit den aktuellsten Rechtstexten versorgt. Egal was Sie an Rechtstexten benötigen, unsere darauf spezialisierten Anwälte verfassen die Texte genau so wie Sie es brauchen.

Sobald Sie sich auf der Seite rechtstexte24.de das Kontaktformular abgeschickt haben, setzen wir uns schnellstmöglich mit Ihnen in Verbindung. Danach können Sie Ihre Seite mit diesem Plugin ausstatten und ganz einfach über vorgefertigte Shortcodes den gewünschten Rechtstext an der dafür vorgesehenen Stelle platzieren.

Bonus: Im Gegensatz zur herkömmlichen Methode haben Sie bei uns den Vorteil, dass das Plugin völlig automatisch entweder stündlich oder aber auch täglich Ihre Rechtstexte aktualisiert.

Dieses Plugin funktioniert nur im Zusammenhang mit rechtstexte24. Sollten Sie zufällig auf dieses Plugin gestoßen sein, schauen Sie bitte auf die Seite http://rechtstexte24.de.

Benötigt mindestens WordPress 3.0 and PHP 5.

= Funktionsübersicht =
* Shortcodes
* Rechtstext einfach über Button im Editor einfügen
* Menü zur Verwaltung der Daten
* stündliche bzw. tägliche Aktualisierungen
* Leistungsübersicht
* Texte können auch manuell angefordert werden

== Installation ==

= Download =
1. Laden Sie sich das Plugin über die Schaltfläche Download Version X.X.X herunter, entpacken die .zip Datei und platzieren es in Ihrer Wordpress Installation unter dem Ordner „Plugins“.

2. ODER

3. Laden Sie sich das Plugin direkt über Ihre Wordpress Installation herunter indem Sie über das Menü „Plugin“ nach JuraText-Importer für rechtstexte24 suchen. Folgen Sie daraufhin den Installations Anweisungen von Wordpress.

= Aktivierung =
1. Gehen Sie in Ihre Wordpress Installation und wählen Sie den Menüpunkt „Plugins“ aus.

2. Suchen Sie in der vorhandenen Liste nach dem Plugin „JuraText-Importer für rechtstexte24“.

3. Aktivieren Sie das Plugin indem Sie auf die Schaltfläche „Aktivieren“ drücken.

4. Nach erfolgreicher Aktivierung erscheint ein neuer Menüpunkt mit dem Namen „JuraText-Importer“. Klicken Sie einmal darauf.


= Plugin-Menü =
5. Sie sind nun auf der Startseite des Plugins. Über die Reiter unter dem Titel können Sie zwischen den verschiedenen Seiten navigieren.

6. Um Ihre Kundendaten einzugeben, drücken Sie einmal auf den Reiter „Einstellungen“, innerhalb des Plugins.

7. Sie können nun die Ihnen von der sirconic-group mitgeteilten Daten, in die dafür vorgesehenen Felder eintragen.

8. Sobald Sie die Felder ausgefüllt haben, drücken Sie auf die Schaltfläche „übernehmen“ um die Einstellungen zu speichern.

9. Die Aktualisierungsrate auf der rechten Seite ist optional und standardmäßig auf stündlich eingestellt.

10. Nachdem Sie auf übernehmen gedrückt haben, werden innerhalb der darauffolgenden 10 Sekunden die Rechtstexte in Ihre Webseite geladen. Sie können dies natürlich auch manuell machen, indem Sie auf die Schaltfläche „Texte anfordern“ klicken.

11. Um einen Rechtstext nun darzustellen, kopieren Sie entweder den dafür vorgesehenen Shortcode (befindet sich unter dem Reiter „Leistungen“) und platzieren ihn auf der dafür vorgesehenen Seite, oder Sie gehen auf die Seite selbst und drücken auf das kleine JuraText-Importer Logo im Editor und geben Ihren gewünschten Rechtstext ein. Der Shortcode wird dann automatisch für Sie zusammengesetzt.

== Frequently Asked Questions ==

= Wie für das Programm anmelden? =

Gehen Sie auf rechtstexte24.de, lesen sich bitte die Seite durch und füllen bei Interesse das Formular aus.
Wir erhalten dann eine Benachrichtigung mit Ihren Daten und werden uns schnellstmöglich darum kümmern.
